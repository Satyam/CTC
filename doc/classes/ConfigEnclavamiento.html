ConfigEnclavamiento Class

Defined in: js/enclavamientos.js:385
Module: configEnclavamiento
Parent Module: CTC

Provee la información de un enclavamiento.

Las propiedades de cada entrada corresponden a cada tipo de enclavamiento salvo la propiedad tipo que es común a todas ellas. Los valores de cada entrada se le pasa como argumento al constructor de cada tipo de enclavamiento. La información sobre estos parámetros se encuentra en la documentación para cada uno de estos enclavamientos.

celdas

Object

Defined in js/sector.js:169

Objeto conteniendo la configuración de cada celda del sector. Cada celda estará indexada por su coordenada X,Y y contendrá un objeto de clase ConfigCelda. No es necesario proveer entradas para celdas que no contienen vías. Las celdas se puede enumerar en cualquier orden.

Example:

{
                      "celdas": {
                        "6,5": {
                          "tipo": "linea",
                          "desde": "W",
                          "hacia": "SE"
                        },
                        ...
                      }
                    }
